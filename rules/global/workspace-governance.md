# Workspace Governance

1. **Separação de Preocupações**: Código de produto nunca deve residir no Henrico-Agent-OS.
2. **Prioridade de Memória**: O arquivo `ai-memory/` de cada projeto é a fonte de verdade suprema.
3. **Padrão de Skill**: Toda nova capacidade deve seguir o template oficial de `SKILL.md`.
4. **Curação Obrigatória**: Ferramentas externas devem passar pelo workflow de curadoria antes de serem oficializadas.
