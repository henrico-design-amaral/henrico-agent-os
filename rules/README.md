# Rules System

Este diretório contém as leis fundamentais que regem o comportamento dos agentes no Henrico-Agent-OS. Diferente de skills (capacidades), as rules são restrições e princípios inegociáveis.

## Estrutura
- `/global`: Regras que se aplicam a todos os projetos.
- `/project-types`: Regras específicas para tipos de projeto (Web, CLI, Research).
- `/git`: Governança de versionamento e limites de repositório.
- `/security`: Proteção de dados e auditoria de ações.
- `/design`: Princípios estéticos e bloqueio de clichês.
- `/content`: Tom de voz e qualidade de escrita.
