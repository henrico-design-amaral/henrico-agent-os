# Implementation Guide


## Repositório sugerido no GitHub

```txt
henrico-design-amaral/henrico-agent-os
```

## Passo a passo

1. Criar repo local.
2. Copiar esta estrutura.
3. Fazer primeiro commit.
4. Criar repositório no GitHub.
5. Conectar remote.
6. Limpar skills antigas dos projetos.
7. Migrar material útil para `external/`.
8. Adaptar apenas o que for realmente útil.
9. Promover skills oficiais para `skills/`.

## Instalação seletiva

Não copie o repositório inteiro para dentro dos projetos.

Para um projeto específico, copie apenas a skill necessária ou aponte o agente para esta biblioteca central.
