# DECISIONS — Henrico Agent OS

## DEC-011 — HenricoOPS substitui o Agent OS como autoridade única

**Data:** 2026-07-22  
**Status:** vigente

### Decisão

O HenricoOPS é o único sistema operacional do ecossistema. O conteúdo metodológico do Agent OS foi incorporado em `henricoops/method/agent-os/`.

### Consequências

- O repositório `henrico-agent-os` é legado de transição.
- Nenhum conteúdo canônico novo deve ser criado aqui.
- O registry de agentes do HenricoOPS prevalece sobre qualquer lista deste repositório.
- As decisões históricas abaixo permanecem preservadas como registro, mas não podem contradizer a arquitetura vigente.
- O arquivamento definitivo depende de auditoria de dependências e aprovação humana.

---

## Decisões históricas preservadas

As decisões DEC-001 a DEC-010 abaixo pertencem ao período anterior à consolidação. Elas continuam úteis como histórico, mas referências a este repositório como autoridade única estão superseded pela DEC-011.
