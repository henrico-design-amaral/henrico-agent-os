# Design System Generation Workflow

Este workflow orienta a criação e evolução de Design Systems locais em projetos de produto.

## Passos Principais
1. **Curadoria de Referências**: Uso de `design-reference-curator` para buscar inspiração premium.
2. **Especificação North Star**: Geração do `DESIGN.md` via `design-md-generator`.
3. **Validação Estética**: Auditoria contínua via `anti-generic-ui-reviewer`.
4. **Exportação de Tokens**: Sincronização de tokens de design com o código real (Tailwind/CSS Vars).
