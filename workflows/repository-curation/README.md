# Repository Curation Workflow

Este workflow define o processo de entrada de ferramentas externas, MCPs e recursos no ecossistema de Henrico.

## Passos Principais
1. **Identificação**: Adição à `registry/tools-watchlist.md`.
2. **Auditoria de Segurança**: Uso de `agent-security-auditor` para analisar riscos da nova ferramenta.
3. **Adaptação**: Criação de uma versão local ou "Skill" que envolva a ferramenta externa.
4. **Oficialização**: Movimentação para `/skills` ou `/mcps` oficiais após validação de Henrico.
