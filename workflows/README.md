# Workflows System

Os workflows definem processos estruturados para tarefas comuns. Eles servem como orquestradores que utilizam múltiplas skills para atingir um objetivo complexo.

## Estrutura
- `/planning`: Processo de fatiamento e especificação.
- `/tdd`: Ciclo de desenvolvimento orientado a testes.
- `/review`: Auditoria de código e design.
- `/repository-hygiene`: Limpeza e manutenção de repositórios.
- `/portfolio`: Atualização e evolução do portfólio.
- `/design-system`: Geração e manutenção de tokens visuais.
- `/long-running-projects`: Gestão de contexto em projetos longos.
