# Commands System

Os comandos são atalhos para executar workflows e skills de forma rápida e padronizada via CLI ou interface de agente.

## Comandos Disponíveis
- `/plan`: Inicia o workflow de planejamento.
- `/review`: Inicia o workflow de revisão técnica e estética.
- `/audit`: Executa auditoria de memória e repositório.
- `/secure`: Verifica vazamentos de dados e vulnerabilidades.
- `/cleanup`: Executa limpeza de arquivos e higiene de repo.
