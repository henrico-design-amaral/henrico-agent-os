# Tools Watchlist

Este documento registra ferramentas externas, frameworks e MCPs candidatos à integração no Henrico-Agent-OS.

> [!IMPORTANT]
> Nenhuma ferramenta desta lista deve ser instalada ou tratada como dependência oficial sem passar pelo processo de curadoria obrigatório.

## Regra de Curadoria de Ferramentas Externas

Ferramentas externas só entram no sistema depois de cumprir as seguintes etapas:

1. **Revisão de Fonte**: Analisar a procedência e a manutenção do repositório.
2. **Revisão de Licença**: Garantir compatibilidade com o uso pessoal e profissional de Henrico.
3. **Revisão de Segurança**: Auditoria via `agent-security-auditor` para identificar riscos.
4. **Teste Local Controlado**: Execução em ambiente sandbox isolado.
5. **Adaptação**: Conversão da funcionalidade para o padrão de `SKILL.md` de Henrico.
6. **Registro Oficial**: Inclusão nos índices do `registry/`.
