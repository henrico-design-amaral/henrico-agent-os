# External Sources Registry

Fontes externas usadas como referência.

Nenhuma fonte externa deve ser copiada cegamente para `skills/`.

Fluxo obrigatório:

1. armazenar original em `external/`
2. revisar segurança e licença
3. extrair padrões úteis
4. adaptar em `adapted/`
5. promover para `skills/` somente após limpeza
