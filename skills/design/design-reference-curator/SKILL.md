# Design Reference Curator

## Propósito
Manter um repositório curado de referências visuais, componentes de elite e padrões de UX premium para inspirar a criação de interfaces.

## Quando usar
- Durante a fase de brainstorming de um novo produto.
- Ao atualizar o `registry/design-md-sources.md`.
- Para validar se uma solução proposta é "estado da arte".

## Quando não usar
- Para busca genérica de imagens.
- Sem um critério claro de "Visual Excellence".

## Entradas esperadas
- URLs de portfólios, bibliotecas de UI ou sites de referência.
- Descrição do problema de interface a ser resolvido.

## Processo
1. Analisar a referência externa contra os padrões de qualidade de Henrico.
2. Extrair os princípios fundamentais da referência (ex: uso de glassmorphism, tipografia bold).
3. Adicionar ao registro local com metadados de "por que isso é bom".

## Saída esperada
- Novos itens no `registry/design-md-sources.md`.
- Snippets de CSS ou Tailwind inspirados na referência.

## Critérios de sucesso
- Disponibilidade de pelo menos 3 referências de alto nível para cada novo componente solicitado.

## Limites de segurança
- Não baixar arquivos binários suspeitos.
- Respeitar direitos autorais (usar apenas como inspiração/referência).
