# Instincts System

Os instincts são heurísticas de decisão rápidas que os agentes devem internalizar. Diferente de rules (restrições) ou workflows (processos), os instincts são "intuições" técnicas baseadas no gosto e nos padrões de Henrico.

## Estrutura
- `portfolio-instincts.md`: Decisões sobre posicionamento e impacto.
- `design-instincts.md`: Decisões sobre estética e UX.
- `git-instincts.md`: Decisões sobre versionamento.
- `content-instincts.md`: Decisões sobre texto e editorial.
